🏥 DocEase

DocEase is a simple web app I built to make managing doctor-patient details easier and more organized. The idea was to create something clean and practical that can handle basic medical records without any confusion.

---

✨ What it does

- Keeps track of patient details
- Lets doctors view and manage records
- Generates basic medical reports (PDF)
- Simple interface so anyone can use it easily

---

🛠️ Tech I used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js

---

📁 Project Structure

DocEase/
│── backend/
│── frontend/

---

⚙️ How to run it

1. Clone the repo

git clone https://github.com/madhvisharma802/DocEase.git

2. Go to backend

cd backend

3. Install dependencies

npm install

4. Start the server

node server.js

5. Open the frontend files in your browser
